## 📌 Project Summary

This is the capstone project for Group 15 of the Novara cohort at TS Academy. The project is a planets web application built with React that fetches and displays data about planets from an external API. It includes a hero section, a planets showcase, a video section, a contact form, and a responsive layout across all screen sizes.

## ⚙️ How to Run Locally

### Prerequisites
- Node.js installed on your machine
- Git installed

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/[your-username]/[repo-name].git
   ```

2. **Navigate into the project folder**
   ```bash
   cd [repo-name]
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

5. **Open in browser**
   ```
   http://localhost:5173
   ```

## 🔄 Git Workflow

- All work is done on separate feature branches
- Branch naming convention: `feature/component-name` (e.g. `feature/hero-section`)
- Contributions are submitted through Pull Requests
- The person who opens a PR does **not** review or merge their own PR
- All PRs must be reviewed and approved before merging into `develop`
- `main` is the production branch — only merged into when the project is ready to deploy
